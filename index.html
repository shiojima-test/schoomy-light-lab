やり方：各レッスンの「▶ このコードを試す」→ 下のコード欄で「実行」→ 結果を見る → 「変えてみよう」に挑戦。

レッスン1：print（表示する）

print("こんにちは")
print(500)
解説：print( ) はカッコの中身を画面に表示する命令。文字は " " で囲む。数字はそのまま書く。
変えてみよう：自分の班の番号と「〇班です」を表示してみよう。

レッスン2：変数（値に名前をつける）

madogawa = 750     # 窓側の明るさ
rokagawa = 300     # 廊下側の明るさ
print(madogawa - rokagawa)   # 差
print(madogawa / rokagawa)   # 何倍か
解説：名前 = 値 で変数を作る。# から後ろはメモ（コメント）。- は引き算、/ は割り算。
変えてみよう：値を自由に変えて「何倍か」がどう変わるか確かめよう。

レッスン3：リスト（値をまとめて入れる）

values = [512, 530, 498, 505, 521]
print("データの数:", len(values))
print("平均:", sum(values) / len(values))
print("最大:", max(values), " 最小:", min(values))
解説：[ ] がリスト。たくさんの数をまとめて扱える。len＝個数、sum＝合計、max/min＝最大/最小。
変えてみよう：リストに値を2つ追加して、平均がどう変わるか見てみよう。

レッスン4：for（くり返す）

values = [512, 530, 498]
for v in values:
    print("計測値:", v)
解説：for v in values: は「リストから1つずつ取り出して v と呼び、字下げした行をくり返す」。字下げ（スペース4つ）が必須。
変えてみよう：くり返しの中で v * 2 を表示してみよう。

レッスン5：if（条件で分ける）

akarusa = 450
if akarusa >= 500:
    print("明るい")
else:
    print("暗い")
解説：if 条件: が成り立てば上、成り立たなければ else: の下が実行される。>= は「以上」。
変えてみよう：akarusa の値をいろいろ変えて、表示が切り替わるのを確認しよう。
