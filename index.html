レッスン1：print（表示する）

print("こんにちは")
print(500)
解説：print( ) はカッコの中身を画面に表示する命令。文字は " " で囲む。数字はそのまま書く。
変えてみよう：自分の名前を表示してみよう。

レッスン2：変数（値に名前をつける）

madogawa = 750     # 窓側の明るさ
rokagawa = 300     # 廊下側の明るさ
print(madogawa - rokagawa)   # 差
print(madogawa / rokagawa)   # 何倍か
解説：名前 = 値 で変数を作る。# から後ろはメモ（コメント）。- は引き算、/ は割り算。
変えてみよう：値を自由に変えて「何倍か」がどう変わるか確かめよう。

レッスン3：リスト（値をまとめて入れる）

values = [512, 530, 498, 505, 521]
print(values)        # リスト全体を表示
print(values[0])     # 1件だけ取り出す（0番目）
解説：[ ] がリスト。たくさんの値をまとめて入れられる。values[0] のように番号（0から）で1件だけ取り出せる。
変えてみよう：values[2] は何番目の値？　いろいろ変えて確かめよう。

レッスン4：for（1件ずつ取り出す）

values = [512, 530, 498, 505, 521]
for v in values:
    print(v)         # 1件ずつ、一気に表示
解説：for v in values: は「リストから1つずつ取り出して v と呼び、字下げした行をくり返す」。5件が一気に表示される。字下げ（スペース4つ）が必須。
変えてみよう：print(v * 2) にすると？

レッスン5：if（1件ずつ判定する）

values = [512, 530, 498, 505, 521]
for v in values:
    if v >= 500:
        print(v, "明るい")
    else:
        print(v, "暗い")
解説：レッスン4のforの中にif文を入れると、1件ずつ判定できる。>= は「以上」。500以上かどうかで表示が分かれる。
変えてみよう：しきい値の500を変えると、判定はどう変わる？

レッスン6：CSVファイルを読み込んで表示する

解説：ここから本物のオープンデータCSV（上で選んだファイル）を扱う。ファイルを開いて、1行ずつ読み、値をリストに入れて表示する。レッスン3〜4のリスト・forがここで生きる。
変えてみよう：上のドロップダウンでデータを変えて、中身がどう変わるか見よう。

レッスン7：CSVのデータで 個数・合計・最大・最小・平均

解説：読み込んだリストから len（個数）・sum（合計）・max/min（最大/最小）・平均を求める。データ分析の基本の集計。
変えてみよう：データを変えて、平均がどう変わるか比べよう。
