# 教室の光環境ラボ v1.0

スクーミー（オレンジボード）で計測した明るさデータを、ブラウザ上のPython環境で分析する高校情報科向け学習システム。

## 構成
- `index.html` … 学習ページ（Pyodide搭載。アカウント・インストール不要）
- `data/demo1〜9.csv` … 練習用デモデータ（教室9地点 A〜I を模擬）
- `data/group1〜9.csv` … 各班の実測データ（ページからアップロードで上書き）

## CSV仕様
`timestamp,value`（ヘッダあり、1秒間隔×3分目安、明るさセンサー生値）

## 授業での使い方
1. 先生がアップロードキー（GitHubトークン）付きURLを配布: `index.html?token=XXXX`
2. 1時間目: デモデータでPython基礎（読み込み→集計→グラフ）
3. 2時間目: スクーミーブロックエディタで計測 → シリアル出力を貼り付けてアップロード
4. 3時間目: 9地点比較・20:1判定・考察

## 対応
情報Ⅰ (3)データ分析 / (4)アルゴリズムとプログラム。学校環境衛生基準（照度500lux・比20:1）を題材化。
