# 教室の光環境ラボ v1.5

スクーミー（オレンジボード）で計測した明るさデータを、ブラウザ上のPython環境で分析する高校情報科向け学習システム。

## 構成
- `index.html` … 学習ページ（Pyodide搭載。アカウント・インストール不要）
- `data/kofu_*.csv` … パート2用の公開データ教材（下記出典）
- `data/demo1〜9.csv` … STEP3練習用デモデータ（教室9地点 A〜I を模擬）
- `data/group1〜9.csv` … 各班の実測データ（ページからアップロードで上書き）

## パート2教材データの出典
気象庁「過去の気象データ検索」（甲府地方気象台の観測値）より作成。
https://www.data.jma.go.jp/stats/etrn/index.php

| ファイル | 内容 |
|---|---|
| kofu_hare.csv | 2026-06-01（晴, 日照12.7h）時別 全天日射量 MJ/㎡（欠測時間帯は0扱い） |
| kofu_kumori.csv | 2026-06-04（曇, 日照0.1h）同上 |
| kofu_1gatsu.csv | 2026年1月 日別 日照時間 h（31日分） |
| kofu_6gatsu.csv | 2026年6月1〜20日 日別 日照時間 h |
| kofu_nazoA.csv | 【先生用・なぞの答え】2026-06-03「大雨後晴時々曇」午前大雨→午後急回復 |
| kofu_nazoB.csv | 【先生用・なぞの答え】2026-06-20「雨時々曇」一日中弱い光 |
| kofu_nazoC.csv | 【先生用・なぞの答え】2026-06-18「雨時々晴一時曇、雷を伴う」朝雨→昼快晴→15時以降雷雨で急落 |

## CSV仕様
`timestamp,value`（ヘッダあり）。時別=1〜24時、日別=1〜31日、班の実測=1秒間隔×3分目安。

## 授業での使い方
1. 先生がアップロードキー（GitHubトークン）付きURLを配布: `index.html?token=XXXX`
2. STEP1: Python基礎＋気象庁の公開データで分析の基礎（可視化/比較/ばらつき/前処理/シミュレーション/推理）
3. STEP2: スクーミーブロックエディタで教室9地点を計測 → アップロード
4. STEP3: 実データで9地点比較・学校環境衛生基準（500lux・20:1）診断・考察
